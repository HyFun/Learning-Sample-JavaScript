网页主要由文字、图像和超链接等元素构成，当然，除了这些元素，网页中还可以包含音频，视频以及Flash等。

2008年，大名鼎鼎的互联网巨头Google公司发布了它的首款浏览器，chrome浏览器。

常见的浏览器内核

IE Trident IE、猎豹安全、360极速浏览器、百度浏览器
firefox Gecko
Safari webkit
Chrome chromeium|Blink blink其实是webkit的分支，大部分国产浏览器最新版本都采用Blink内核二次开发

扩展阅读：
移动端的浏览器内核主要说的是系统内置浏览器的内核。
Android手机而言，使用率最高的就是webkit内核，大部分国产浏览器宣称的自己的内核，基本上也是属于webkit二次开发
IOS以及WP7平台上，由于系统原因，系统大部分自带浏览器内核，一般都是Safari或者IE内核Trident的
